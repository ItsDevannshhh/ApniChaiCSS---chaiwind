Syntax

Every class follows the same pattern:

chai - property - value
chai-p-20

padding: 20px

chai-bg-red

background-color: red

chai-text-center

text-align: center

chai-rotate-45

transform: rotate(45deg)

Installation

Pick whichever way fits your project.

📦 npm

# install npm install apnichaicss # use in your JS import { applyChaiStyles } from "apnichaicss"; applyChaiStyles();
npmjs.com/package/apnichaicss →

🌐 CDN

<!-- paste in <head> --> <script type="module" src="https://unpkg.com/ apnichaicss/index.js" ></script> <!-- then just use classes --> <div class="chai-p-20 chai-bg-red"> Hello ☕ </div>

Note: npm requires a bundler (Vite, webpack, etc.). CDN auto-runs in the browser — no extra function call needed.
